== Description ==

We Are Team is a Gutenberg WordPress plugin that provides an easy and elegant way to showcase a group of people on your website. Designed with simplicity in mind, this plugin integrates seamlessly with Gutenberg, allowing you to create visually appealing team sections for your blog.

- **Gutenberg Integration:** Effortlessly add and manage team members using the Gutenberg editor.
- **Visually Appealing Design:** Showcase your team members in a visually appealing and user-friendly way.
- **Personal Touch:** Add a personal touch to your website by highlighting the people behind your project or organization.

== Frequently Asked Questions ==

= How to install the plugin =

1. Upload the plugin files to the `/wp-content/plugins/werteam` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= How to use it? =

1. Open a page or post in the Gutenberg editor.
2. Add a new block and search for "We Are Team."
3. Start adding team members, including their names, roles, and images.
